# template-lib
Nodejs Template for libraries

This package includes the following:  
- ES-Lint
- Babel
- Mocha & chai
- Coverage Report (nyc)


## ES-Lint
> npm run lint

## Babel
> npm run build  

This command removes dist/ folder and builds it from the contents in src/ folder

## Mocha & Chai (Testing)
> npm run test

## Coverage Report
> npm run coverage
